<h1>URL Shortener</h1>

<p>
This is a simple URL Shortener built using Streamlit and Python. The app takes 
a long URL from the user and converts it into a shorter and easier-to-share link 
using the TinyURL API.
</p>

<p>
I built this project to practice working with APIs in Python and to understand 
how HTTP requests work in a small real-world example. It also helped me explore 
how quickly Streamlit can be used to build simple web apps.
</p>

<h3>Features</h3>
<ol>
  <li>Enter any long URL and convert it into a short link</li>
  <li>Uses TinyURL API for link generation</li>
  <li>Instant result display</li>
  <li>Simple and minimal interface</li>
</ol>

<h3>Tech Stack</h3>
<ul>
  <li>Python</li>
  <li>Streamlit</li>
  <li>Requests</li>
  <li>TinyURL API</li>
</ul>

<h3>How to Run</h3>
<ol>
  <li>Clone this repository</li>
  <li>Install the required packages:</li>
</ol>

<pre><code>pip install streamlit requests</code></pre>

<h3>Run the app:</h3>

<pre><code>streamlit run app.py</code></pre>

<h2>Why This Project?</h2>

<p>
The purpose of this project was to learn how to work with APIs in Python and see 
how external services can be integrated into a small application. It’s a simple 
project, but it shows how a few lines of code can turn a basic idea into a 
working web tool.
</p>
